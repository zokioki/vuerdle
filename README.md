# vuerdle

A Vue port of [Wordle](https://www.powerlanguage.co.uk/wordle/).

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
